## データ

### tasks

| カラム   | 型     | 必須項目 | 備考                                          |
| -------- | ------ | -------- | --------------------------------------------- |
| priority | number | 必須     | 優先度、1から5段階、数字で表記、5が最高優先度 |

## 画面一覧

### タスク一覧画面

- タスク名の右側に「優先度」という項目をつくります。
- 「優先度」はカラムデータの priority に含まれる number 型の数値になります。
- 「優先度」は数字の1から5までの5段階、5が最高優先度です。
- 「優先度」の数字のバックグラウンドは、1が緑、2が黄、3がオレンジ、4が赤、5が紫になります。
- 「優先度」も他のタスクと同様に編集可能にしてください。
- 「優先度」の入力は必須です。
